# Helm Repositories

HelmRepository CRDs for Flux to access Helm charts.

## Service

- **HelmRepository resources**: Flux CD repository definitions
- **Namespace**: flux-system

## Secrets

None

## Configuration

Repositories configured:

- Longhorn
- Jetstack (cert-manager)
- Bitnami (PostgreSQL, Redis)
- Grafana
- Prometheus Community
- Authelia
